# SELDEX

## Overview

SELDEX is a modern web application built with Next.js that drives "First Time Right" (FTR) outcomes for Loan Agents and Direct Selling Agents (DSA). The platform leverages Intelligent Document Processing (IDP) and Data Governance to enable audit-ready loan processing.

## Features

- **Intelligent Document Processing**: Automated processing of loan documents with high accuracy
- **Data Governance**: Comprehensive data management and compliance framework
- **Audit-Ready**: Built-in compliance and audit trail capabilities
- **Modern UI**: Responsive design with smooth animations using Framer Motion
- **Performance Optimized**: Built on Next.js 16 with TypeScript

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org) 16.1.5
- **Language**: [TypeScript](https://www.typescriptlang.org)
- **UI Library**: React 19.2.3
- **Styling**: [Tailwind CSS](https://tailwindcss.com) 4
- **Animations**: [Framer Motion](https://www.framer.com/motion) 12.29.2
- **Icons**: [Lucide React](https://lucide.dev) 0.563.0
- **Utilities**: clsx, tailwind-merge
